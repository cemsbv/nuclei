PileCoreÔÉÅ

PileCore aims to give you the full pile calculation utilities without an engineer in the loop. This means that calculations must be able to run from start to finish from a set of given parameters. Pilecore is able to determine the tipping point between positive and negative friction and will optimize pile groups automatically. The complete API documentation can be accesses here.

Please note that you need a NUCLEI account to call one of our endpoints. You can sign up here to get your free access to PileCore! For this example we set the account information in our environment. If you are not confident to reproduce that nuclei will ask you to provide the user token when calling the endpoint.

Lets show you how to use nuclei and access the PileCore API.

In [1]: from time import sleep

In [2]: import pandas as pd

In [3]: from nuclei.client import NucleiClient

# set app name
In [4]: APP = "PileCore"

# create session
In [5]: client = NucleiClient()

Next we define the schema to send to the endpoint.
