CPT Core

The CPT Core is the first developed API that has been widely used by CRUX employees and external clients in the past years. It consists of a fully automated soil classification based on a convolutional neural network, The complete API documentation can be accesses here.

Please note that you need a NUCLEI account to call one of our endpoints. You can sign up here to get your free access to CPT Core! For this example we set the account information in our environment. If you are not confident to reproduce that nuclei will ask you to provide the user token when calling the endpoint.

Lets show you how to use nuclei and access the CPT Core API.

In [1]: import os

In [2]: from pygef import read_cpt

In [3]: import numpy as np

In [4]: from nuclei.client import NucleiClient

# set app name
In [5]: APP = "CPT Core"

# create session
In [6]: client = NucleiClient()

Next we use the cpt parser of pygef to parse a cpt and create the request body. This body is used to call the “/plot” endpoint of the CPT Core with call_endpoint().

# Parse a CPT file with pygef.
In [7]: path_cpt = os.path.join(
   ...:     os.environ.get("DOC_PATH"), "_static/data/cpt.gef"
   ...: )
   ...: 

In [8]: cpt = read_cpt(path_cpt)

# create the body to send to the gef-model endpoint.
In [9]: schema = {
   ...:     "cpt_object": {
   ...:         "name": cpt.alias,
   ...:         "x": cpt.delivered_location.x,
   ...:         "y": cpt.delivered_location.y,
   ...:         "ref": cpt.delivered_vertical_position_offset,
   ...:         "index": np.array(
   ...:             cpt.data["depthOffset"], dtype=float
   ...:         ),
   ...:         "qc": np.array(cpt.data["coneResistance"], dtype=float),
   ...:         "fs": np.array(cpt.data["localFriction"], dtype=float),
   ...:     },
   ...:     "img_encoding": "base64",
   ...: }
   ...: 

# call the gef-model endpoint with nuclei
In [10]: plot = client.call_endpoint(APP, "/plot", schema)

The “/classify” endpoint allows you the access the data of the graph above. Please note that the call_endpoint() will transform the responds to python types by default. This means that for example a polars DataFrames are transformed from json back to the DataFrame.

# call the gef-model endpoint with nuclei
In [11]: responds = client.call_endpoint(APP, "/classify", schema)
